The conceptual model & E/R diagram

From the chapters "I materiali" and "Le tecniche" it is possible to extract the first entities and relations that could formalize the concepts of materials and techniques in the artworks.

The first step was the creation of a conceptual model which tries to define the main elements and how they are related. Since materials and techniques are still at the level of denotative description, some elements as "Signature" which can have a different specific material or technique have been added to the ontology.

Those relations have been formalized in the following E/R diagram

Ontology Design and Implementation
