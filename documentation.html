Ontology Design and Implementation

The conceptual model

From the chapters I materiali and Le tecniche it is possible to extract the first entities and relations that could formalize the concepts of materials and techniques in the artworks.

The first step was the creation of a conceptual model which tries to define the main elements and how they are related. Since materials and techniques are still at the level of denotative description, they are related to all the elements, as "Signature" or "Frame", which sometimes are added through a different technique or material from the one largely used for the artwork.

The Ontology

Starting from the very first theorization, then we proceded by the implementation of the actual ontology.

The ontology was designed on Protégé, an open-source ontology editor. Classes, object properties, data properties, named individuals, annotation properties, general axioms and namespace declarations are described in the documentation.

The documentation is accessible as an HTML page thanks to Live OWL Documentation Environment (LODE), which allows to automatically extract and visualize each element of an onotlogy from the OWL file.

Data Extraction and Dataset

Starting from Zeri's catalogue of paintins, an automatic extraction was performed on the text in order to define the main elements: title, author, material and technique.

The steps to extract the data can be summed as follows:

  1. Cleaning and text preparation: provide a clean text, in a machine-readable format
  2. Paragraphs and section: automatic division of the text into titles and paragraphs. In this way it will be easier to define where to search information. In this case, the division has been performed through Regular Expression (RE) but other methods can be used according to the internal organization of the text (for example, nlp tools as spacy provide a paragraph division).
  3. Definition of titles and authors: after the identification of titles, through Names Entity Recognition it was possible to extract names of authors, while name of artworks where extracted through RE.
  4. Extraction of materials and techniques: in order to perform an automatic extraction we used as reference the two skos vocabularies:
    • Thesaurus per la definizione della materia per le opere e gli oggetti d'arte
    • Thesaurus per la definizione della tecnica per le opere e gli oggetti d'arte
    Since both of them where in italian, it was necessary to translate them in English. This operation was automated too through the python library deep translator

In order to test as much classes as possible from the ontology, for 35 artworks additional information (as signature, inscription, support etc.) where manually extracted from the reference text and added to the dataset.

Knowlede Graph and results

The intersection of the automatic and the manually extracted data has been used in order to populate the knowledge graph (python code available here)

Now we can translate the Competency Questions in SPARQL queries that can be performed using OM&TA ontology on a sparql endpoint.

The data have been loaded on a graph database (Blazegraph) on which the queries have been performed to get the results.

CQ1

Which technique has been used in arwork X?

PREFIX mat: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat#>
PREFIX mat-r: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat/resource/>

SELECT ?techniqueLabel WHERE {
mat-r:P007 mat:hasTechnique ?technique.
?technique rdfs:Literal ?techniqueLabel
}

oil, tempera

CQ2

From which material is artwork Y made of?

PREFIX mat: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat#>
PREFIX mat-r: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat/resource/>

SELECT ?techniqueLabel ?materialLabel WHERE {
mat-r:P007 mat:hasTechnique ?technique.
?technique mat:isPerformedOn ?material;
rdfs:Literal ?techniqueLabel.
?material rdfs:Literal ?materialLabel
}

1 - techniqueLabel: oil; materialLabel: wood
2 - techniqueLabel: tempera; materialLabel: wood.

CQ3

Which are the most common materials on which is performed technique Z?

PREFIX mat: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat#>

SELECT ?materialLabel (COUNT(?material) as ?materialCount) WHERE {
?technique mat:isPerformedOn ?material
FILTER(contains(str(?technique), "oil"))
?material rdfs:Literal ?materialLabel
}
GROUP BY ?materialLabel
ORDER BY DESC(?materialCount)

1 - material: canvas; materialCount: 17
2 - material: wood; materialCount: 7
3 - material: paper; materialCount: 2

CQ4

Which are the most common materials in art?

PREFIX mat: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat#>

SELECT ?materialLabel (COUNT(?material) as ?materialCount) WHERE {
?technique mat:isPerformedOn ?material.
?material rdfs:Literal ?materialLabel
}
GROUP BY ?materialLabel
ORDER BY DESC(?materialCount)
LIMIT 3

1 - materiaLabel: wood; materialCount: 22
2 - materialLabel: canvas; materialCount: 18
3 - materialLabel: marble; materialCount: 15

CQ5

Which artworks have changed support over time?

PREFIX mat: <http://www.semanticweb.org/bianc/ontologies/2023/1/mat#>

SELECT DISTINCT ?artworkLabel WHERE {
?artwork a mat:Artwork;
mat:hasSupport ?currentSupport;
rdfs:Literal ?artworkLabel.
?currentSupport mat:transferredFrom ?originalSupport.
}

artworkLabel: Three Saints: Roch, Anthony Abbot, and Lucy
